Then, the data was splitted into train and test data.

set.seed(354)

head(rows)

split
## [1] 43152
train <- diamonds[1:split,]
test <- diamonds[(split + 1):nrow(diamonds),]

K-fold cross validation was applied to improve the model with k=5.

model <- train(
  price ~ ., diamonds,
  method = "lm",
  trControl = trainControl(
    method = "cv", number = 5,
    verboseIter = TRUE
  )
